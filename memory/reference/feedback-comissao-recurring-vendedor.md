# Comissão recurring perpétua a vendedor terceiro = suicídio financeiro

Quando Wagner negocia comissão pra vendedor (familiar OU terceiro), **NUNCA aceitar % MRR perpétuo sem decay**. Padrão de mercado SaaS B2B brasileiro:

| Componente | % típico | Lógica |
|---|---|---|
| **Upfront/implantação** | 10-20% | Pago uma vez, fixo |
| **MRR primeiro ano** | 15-25% | Recompensa pela venda inicial |
| **MRR ano 2** | 5-10% | Recurring decay |
| **MRR ano 3+** | 0% | Cliente já é "casa do produto" |
