## Problemas encontrados e soluções

### React + Babel CDN falhou para preview

**Sintoma:** tela em branco, warnings no console, Babel tentando transpilar e falhando silenciosamente.

**Causa:** complexidade de setup React/shadcn via CDN sem bundler; shadcn depende de CSS variables e peer deps que Tailwind CDN não expõe de forma robusta.

**Solução:** abandonar React para preview. Usar HTML puro + Tailwind/Chart.js CDN. Registrado como preferência permanente (ADR 0009, preferences.md).

### Sidebar com 10 itens conflitava com realidade do UltimatePOS

**Sintoma:** primeiro preview mostrava sidebar longa com 10 opções do Ponto WR2. Eliana rejeitou imediatamente — seus clientes já têm sidebars longas por conta dos módulos instalados.

**Solução:** 1 item na sidebar + menu horizontal em abas dentro da página do módulo. Registrado como ADR 0008.
