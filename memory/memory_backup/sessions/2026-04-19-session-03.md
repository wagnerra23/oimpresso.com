## Diagnóstico

Comparando `Modules/Jana`, `Modules/Repair`, `Modules/Project` e `Modules/Manufacturing`:

- **Todos os módulos que aparecem na sidebar têm** `Http/Controllers/DataController.php` com o método `modifyAdminMenu()`.
- Esse método usa a fachada `Menu::modify('admin-sidebar-menu', …)` para injetar itens na sidebar.
- O core UltimatePOS descobre o método automaticamente via middleware `AdminSidebarMenu` — não há registro explícito em evento/provider.
- **PontoWr2 não tinha esse arquivo** — causa raiz do menu ausente.

Confirmações auxiliares:

- Middleware `AdminSidebarMenu` já está na stack de rotas do PontoWr2 (`Http/routes.php` linhas 24 e 126). OK.
- Middleware `ponto.access` (`CheckPontoAccess`) existe e está aliasado no `PontoWr2ServiceProvider`. OK.
- Lang `Resources/lang/pt/ponto.php` já tinha as chaves `menu.*` prontas. OK.

## Correção aplicada

### 1. Criado `Modules/PontoWr2/Http/Controllers/DataController.php`

Seguindo o padrão Jana (dropdown com sub-itens, porque o Ponto tem muitas telas) + Repair (estrutura enxuta do controller).

O controller expõe três métodos que o core UltimatePOS chama por convenção:

- `superadmin_package()` — registra o pacote `ponto_module` no painel Superadmin
- `user_permissions()` — registra 5 permissões (`ponto.access`, `ponto.colaboradores.manage`, `ponto.aprovacoes.manage`, `ponto.relatorios.view`, `ponto.configuracoes.manage`) para o cadastro de Roles
- `modifyAdminMenu()` — injeta o dropdown **"Ponto WR2"** com os 10 sub-itens:
  - Dashboard, Espelho, Aprovações, Intercorrências, Banco de Horas, Escalas, Importações, Relatórios, Colaboradores, Configurações

O dropdown respeita permissões por item: alguns só aparecem para superadmin ou para quem tem a permissão específica (ex.: `ponto.colaboradores.manage`).

Padrão de verificação de ativação (igual ao Jana):

```php
if (auth()->user()->can('superadmin')) {
    $is_ponto_enabled = $module_util->isModuleInstalled('PontoWr2');
} else {
    $is_ponto_enabled = (bool) $module_util->hasThePermissionInSubscription(
        $business_id, 'ponto_module', 'superadmin_package'
    );
}
```

Ordem do menu: **25** (fica entre Reparar = 24 e demais itens de operação).

Ícone do dropdown: `fa fas fa-business-time`.
