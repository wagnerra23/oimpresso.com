# Concorrentes ERPs oficina auto BR — 2026-05-09

> Pesquisa competitiva 2026-05-09 — exploração viabilidade vertical auto. **NÃO é compromisso de construir.** Fontes: site oficial dos concorrentes, Reclame Aqui, LinkedIn, CNPJ.biz/Econodata, blogs setoriais (EV8 Auto, Mercado Veículos, Sindirepa, Onmotor, Doutor-IE, OficinaBrasil), Hotmart, Google Play. Apenas conteúdo público — sem engenharia social ou criação de conta em demos.
>
> **Diferenciais oimpresso usados como contraste:** NFC-e automática a partir de boleto pago (US-RB-044 + US-NFE-002 com 11 templates fiscais L1); Jana IA com memória persistente (laravel/ai 0.6.3 + Meilisearch hybrid + 4 Agents); Multi-tenant Tier 0 isolation (ADR 0093 — irrevogável); Stack moderna Laravel 13.6 + Inertia v3 + React 19 + Tailwind 4; Modules/Repair Kanban drag-drop entregue PR #363 (mesma estrutura serve oficina); Governança formal Constituição v2 (ADR 0094); Cliente piloto real (ROTA LIVRE biz=4) validado 2 anos em outro vertical.
>
> ⚠️ **Calibração de nomes:** dos 10 ERPs hipotéticos do brief (Mecânico/Tecnosistemas, ManagerOS, Auto Manager, Lokoz, Olho Vivo, OficinaMaster, Carros2, Garage/HuntDevs, Skill Auto, Workshop) — **nenhum apareceu como líder real do mercado BR em busca direta**. O mapeamento abaixo usa os concorrentes que **realmente** disputam clientes em pesquisas Google "sistema oficina mecânica" + ranking EV8 Auto + comparativos WSoft/Wüst. Quem o brief chamou de "Auto Manager"/"Garage" provavelmente referia oficina.app (App Garage) e similares — incluídos.

## Onde oimpresso atacaria — 3 wedges concretos

### Wedge 1: NFC-e + NFS-e automáticas a partir de boleto pago
- **Diferencial:** Modules/NfeBrasil entregue com 11 templates fiscais L1 (10 UF + MEI), pipeline boleto-pago→evento→NFCeAutorizada já validado em biz=1 (Wagner WR2 SC) homologação SEFAZ-SC ([auto-mem `project_nfebrasil_estado_2026_05_07.md`](memory/MEMORY.md)).
- **Ataque:** Ultracar tem reclamação pública crônica de NFS-e travada. Oimpresso já provou pipe ponta-a-ponta.
- **Pitch:** "Você não emite NFS-e/NFC-e pelo sistema porque trava? No oimpresso, o boleto pago no Asaas/Inter dispara a NFC-e automática — sem clique. Já validado em homologação SEFAZ."

### Wedge 2: Jana IA com memória persistente — diagnóstico assistido + Q&A faturamento
- **Diferencial:** laravel/ai 0.6.3 + Meilisearch hybrid + 4 Agents (BrainB/Planner/Decomposer/Reviewer) + ContextoNegocio com 3 ângulos faturamento. Já validado em CYCLE-01 prod (Larissa pergunta "quanto faturamos" e recebe resposta correta).
- **Ataque:** Zero concorrente do mapeamento tem IA real. "Inteligência" é palavra de marketing, não LLM com memória.
- **Pitch:** "O dono da oficina pergunta 'qual cliente está atrasado, qual peça vendeu mais em abril' direto no chat e recebe resposta em 3 segundos — sem abrir relatório. Concorrentes te entregam dashboard estático."

### Wedge 3: Stack moderna + multi-tenant Tier 0 + governança formal
- **Diferencial:** Laravel 13.6 + Inertia v3 + React 19 + Tailwind 4 nativo cloud; multi-tenant `business_id` global scope obrigatório (ADR 0093 IRREVOGÁVEL); Constituição v2 (ADR 0094) pública; Modules/Repair Kanban drag-drop entregue (PR #363).
- **Ataque:** Concorrentes legacy (IS2, NeXT, Limersoft) não têm como alcançar; cloud puro novato (Oficina Inteligente) sem track record; Ultracar/Soften com stack desconhecida.
- **Pitch:** "Você quer crescer pra 3 unidades — multi-tenant nativo desde o dia 1. Quer abrir API pra contador? Constituição v2 publica como; nenhum concorrente expõe esse nível de governança."
