- "?" = não encontrado no site público (nunca inventei).
- Stack inferida só quando explícita (ecommerce ativo, integração shopping, multi-loja).
- Endereços/telefones aqui são publicados — sem PII pessoal.

## Notas de pesquisa

### O que aprendi sobre o mercado de varejo de vestuário no PR

**Padrão dominante #1 — IG é o "ecommerce" de fato.**
27 das 28 empresas mapeadas usam Instagram como canal #1 (não site próprio). Cor de Pimenta tem 251k seguidores; Cavaliere tem 260k. Pra essas operações, o IG é literalmente onde rola o orçamento + venda + pós-venda. **Dor clara**: integração IG↔estoque↔NFe↔financeiro é manual (planilha + WhatsApp + caderno).

**Padrão dominante #2 — multi-loja é a norma no high-end.**
Tier 1 tem 5 de 10 com **2+ lojas físicas** (Impelle, Baoo 3, Maria Perfeita 2, Sottile 3) — confirma sweet spot multi-tenant ICP. Tier 2 quase todos single-store (porte menor).

**Padrão dominante #3 — atacado + varejo dual (Cavaliere model).**
Cavaliere Londrina é um padrão importante: **mesma operação atende B2B (atacado de revendedor) + B2C (varejo final)**. Preço/condição/NFe diferentes pra cada canal. ROTA LIVRE (cliente piloto) é puro varejo, mas Cavaliere abre porta pra explorar dual-channel no Modules/Vestuario.

**Padrão dominante #4 — concentração geográfica clara.**
- **Curitiba**: cluster Batel (Impelle, Baoo, Boutique Ideale, Bonaparte, Pátio Batel) = polo high-end PR; Cabral (Jaque Bontorin, Namix legado); Bom Retiro/Centro (Maria Perfeita, Shirley)
- **Maringá**: Avenida Shopping + Av. Cerro Azul = eixo varejo
- **Londrina**: Centro (R. Mato Grosso/Higienópolis/Av. Paraná) — Cavaliere âncora
- **Foz do Iguaçu**: Mons Guilherme + Rui Barbosa (centro turístico) — Sottile com 3 endereços

**Padrão dominante #5 — porte regional + longevidade alta.**
Média de fundação Tier 1 ≈ 2005 (~20 anos). Tier 2 tem outliers de 40+ anos (Rawell PG, Estratosfera Maringá). Empresas longevas = sistema atual = ERP genérico antigo (Bling/Tiny) ou Excel. Receptividade média a high.

### Sinais de dor compartilhada (validação ICP Modules/Vestuario)

1. **IG como balcão de vendas** — sem rastreio funil, sem histórico cliente, sem follow-up automático
2. **Multi-loja sem visão consolidada** (Impelle, Baoo, Maria Perfeita, Sottile)
3. **SKU explosão** (Cor de Pimenta lança semanal; Boutique Ideale mistura perfumaria + moda; Maria Perfeita mistura fem + masc + festa)
4. **Consignação multimarca** (Impelle, Baoo, Cor de Pimenta) — rateio entre marcas é dor de planilha
5. **Atacado + varejo dual** (Cavaliere) — preço/condição/NFe diferenciados
6. **Estoque caro com giro lento no high-end** (Baoo calçado high-end Birman/Repetto = ticket grande, capital parado)
7. **Sazonalidade turismo** (Sottile Foz) — multi-loja com pico em férias, dor de previsão demanda
