## Análise: nosso pricing está caro/barato/certo?

### Starter R$ [redacted Tier 0] (1 func, <200 vendas/m)
- **Concorrente direto:** Bling Cobalto R$ [redacted Tier 0] (anual), Tiny Avance R$ [redacted Tier 0] (anual), Omie Fit grátis (MEI)
- **Veredicto:** **CARO em comparação direta** — 5x Bling. Mas Bling é horizontal e oimpresso é vertical com NFCe automática + foco em comunicação visual + integração Asaas
- **Justificativa pra manter R$ [redacted Tier 0]:** posicionamento vertical (vale o "imposto da especialização"). Mas explicar bem na página: "vs Bling você economiza X horas/m porque não precisa configurar OS-CFOP-cliente do zero"
- **Risco:** churn alto se cliente comparar tabela e ver "5x Bling sem motivo claro"

### Pro R$ [redacted Tier 0] + setup R$ [redacted Tier 0] (5-10 func)
- **Concorrente direto:** Tiny Construa R$ [redacted Tier 0] (anual), Bling Titânio T1 R$ [redacted Tier 0] Conta Azul Avançado R$ [redacted Tier 0]
- **Veredicto:** **competitivo no mensal**, mas **setup é o ponto fraco** — todos os horizontais cobram R$ [redacted Tier 0] de setup
- **Justificativa setup:** migração de dados Delphi/legado + treinamento + customização CFOP/CSOSN (Wagner WR2 sabe que isso é trabalho real)
- **Risco:** "Bling não cobra setup, por que oimpresso cobra R$ [redacted Tier 0]?" — precisa argumento concreto. **Recomendação: oferecer setup R$ [redacted Tier 0] (cliente que NÃO migra de Delphi/Excel) e R$ [redacted Tier 0] (migração completa de WR Comercial / planilhas / outro ERP)**

### Enterprise R$ [redacted Tier 0] + setup R$ [redacted Tier 0] (10-30 func)
- **Concorrente direto:** Tiny Domine R$ [redacted Tier 0] Bling Diamante R$ [redacted Tier 0] Conta Azul Performance R$ [redacted Tier 0] Sankhya R$ [redacted Tier 0]+, Omie Multivarejo R$ [redacted Tier 0]+

## Recomendações de ajuste pricing — TOP 3

### 1. Reduzir setup Pro de R$ [redacted Tier 0] → R$ [redacted Tier 0] (default) com upgrade R$ [redacted Tier 0] só pra "migração complexa documentada"
**Razão:** Bling/Tiny/Conta Azul cobram **R$ [redacted Tier 0] de setup**. Setup R$ [redacted Tier 0] sem diferenciação visível é o atrito #1 na conversão. Defender R$ [redacted Tier 0] só quando cliente migra de Delphi WR Comercial ou ERP concorrente complexo (custo real de migração de dados é alto). Default Pro = R$ [redacted Tier 0] (treinamento 4h + ativação) para reduzir fricção.

### 2. Oferecer 14 dias trial grátis em Starter e Pro (sem cartão de crédito)
**Razão:** Bling, Tiny, Omie todos oferecem **30 dias trial**. Conta Azul oferece **3 dias** (curto, mas existe). oimpresso sem trial = "compre cego" — perde concorrência mesmo que produto seja superior. 14 dias é meio-termo: tempo pra cliente importar 1 produto, emitir 1 NFe-teste, sem o custo operacional de bancar 30 dias. Hostinger não cobra por tenant adicional se for sandbox separado.

### 3. Promoção "12 meses paga 10" no anual + "primeiro mês setup grátis" pro Starter
**Razão:** Bling tem "anual = 2 meses grátis" (R$ [redacted Tier 0] vs R$ [redacted Tier 0]×12=R$ [redacted Tier 0]). Tiny igual. **Padrão de mercado.** oimpresso anual: Starter R$ [redacted Tier 0] (em vez de R$ [redacted Tier 0]), Pro R$ [redacted Tier 0] (em vez de R$ [redacted Tier 0]), Enterprise R$ [redacted Tier 0] (em vez de R$ [redacted Tier 0]). Pro paga setup (R$ [redacted Tier 0]) parcelado em 12x dentro da anualidade = sensação de "setup grátis no anual" sem perder receita.

## Riscos identificados

1. **Asaas é gratuito até infinito** (só taxa por transação). Qualquer mensagem oimpresso que pareça "venda Asaas+ERP integrado" precisa explicar que oimpresso **não substitui Asaas** (taxa de transação é inevitável) — substitui o ERP em volta. Risco de cliente confundir "estou pagando R$ [redacted Tier 0] oimpresso e ainda pago Asaas?"
2. **Bling tem 30 dias trial + zero setup + 5-12x mais barato no entry tier.** Se oimpresso não comunicar valor vertical claramente (CFOP de gráfica configurado, OS de comunicação visual com m², integração com fornecedor de plotter etc), cliente compara tabela de preços e some
3. **Conta Azul cobra mais que Bling mas inclui migração contábil.** oimpresso cobrar setup R$ [redacted Tier 0] sem mostrar "migração de Delphi WR Comercial em N dias" = setup parece arbitrário
4. **Sankhya/Protheus não publicam preço.** Race-to-bottom em SaaS BR pra SMB significa Bling/Tiny puxam pra baixo (R$ [redacted Tier 0]–700/m). Médio-grande continua oligopólio Sankhya/TOTVS (R$ [redacted Tier 0]+/m). oimpresso Enterprise R$ [redacted Tier 0] fica numa "terra-de-ninguém" — abaixo de Sankhya, acima de Tiny Domine. Bom posicionamento se nicho gráfica grande aceitar
5. **Granatum: discrepância de preço público R$ [redacted Tier 0] vs R$ [redacted Tier 0]** (search vs site oficial). Sinal de promoção rotativa — dado a desconsiderar até validar
6. **TOTVS Protheus implantação não-publica** mas estimativas de mercado (Logos/Upduo/RFB) apontam R$ [redacted Tier 0]k–200k+ setup e R$ [redacted Tier 0]+/m. oimpresso Enterprise é **muito mais barato** mas precisa argumentar SLA e suporte de qualidade equivalente
