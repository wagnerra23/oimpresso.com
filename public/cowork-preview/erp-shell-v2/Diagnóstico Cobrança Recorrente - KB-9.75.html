Refino #1 — Fundação + Promoção mockup→canônico

3 frentes paralelas: (1.a) criar os 3 arquivos .jsx + rota, (1.b) teclado canônico (⌘K + J/K + cheat-sheet), (1.c) pílula de status + responsive tabs. Tudo entra junto porque um depende do outro — não dá pra fazer ⌘K sem ter o componente.

Frente 1.b

Teclado canônico

Operador veterano (Eliana) não usa mouse. Atalhos visíveis. Reuso direto dos componentes existentes — não reinventar.

  • ⌘K abre command palette escopada em assinaturas + planos + próximas cobranças. Reuso window.CommandPalette da Caixa.
  • J / K navegam linha-a-linha na lista de assinaturas.
  • ↵ abre drawer de detalhe da assinatura focada.
  • R "Retentar agora" na linha focada (se status = falhou/retentando).
  • P "Pausar" / "Reativar" toggle.
  • E "Editar plano" — abre ChangePlanModal.
  • N "Nova assinatura" — abre drawer.
  • / foca a busca local.
  • Esc fecha drawer/modal.
  • ? abre cheat-sheet flutuante (overlay com lista de atalhos).
