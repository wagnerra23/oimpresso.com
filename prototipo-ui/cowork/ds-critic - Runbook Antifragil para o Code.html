A As 3 propriedades que fazem durar e melhorar
viveMora no git / CI

O método não é um doc que alguém lembra de seguir — é código que roda sozinho a cada PR. Sobrevive a trocas de pessoa, de contexto, de tempo. Sua memória (ADR, Ledger) já persiste assim.

cataracaSó melhora, nunca regride

Cada merge crava um piso novo: cor crua só pode cair, score só pode subir. Se um PR pioraria a métrica, o CI barra. O ganho fica travado — ninguém desfaz sem querer.

aprendeCada escape vira gate novo

Defeito que passou pro seu olho = falha do sistema, não sua. Vira lição (LICOES_CC.md) e um teste automático. Aquela classe nunca mais passa. Antifrágil: erro o deixa mais forte.

Seu protocolo já tem metade disto

LICOES_CC.md (erros a não repetir) e proibicoes.md já são o corpus de aprendizado — a metade humana. A evolução é só fechar o laço na máquina: toda lição nova também gera um gate. O que hoje você lembra de não repetir, amanhã o CI impede de repetir.

C A auto-avaliação — o crítico critica a si mesmo

"Auto-avaliativo" de verdade = o sistema mede a própria saúde, não só a das telas. Periodicamente o ds:critic --meta pergunta: os gates ainda pegam? (taxa de escape caindo?) · há falso-positivo irritando à toa? · a cobertura das 5 lentes está completa? Se a taxa de escape parar de cair, o sistema sinaliza que precisa de lente nova. Assim ele não só melhora as telas — melhora a si mesmo.

Resumo: por que isto sobrevive ao tempo

Vive no CI (não depende de memória humana) · cataraca (ganho nunca regride) · aprende (cada escape vira gate, cobertura → 100% assintótico) · auto-avalia (mede a própria taxa de escape). Quatro propriedades, e seu protocolo já carrega as raízes de todas — ADR, Ledger, LICOES_CC, proibicoes. Não é sistema novo: é o seu, fechado em laço.

Encaixe no protocolo: isto É a F1.5 [CD] formalizada. Artefatos (critique-score.json, ≥80) já nomeados no PROTOCOL. Aprendizado usa LICOES_CC.md + proibicoes.md existentes. Soberania honrada: [CC] propõe, [W] decide, [CL] numera/comita — nada na constituição muda sem [W].
