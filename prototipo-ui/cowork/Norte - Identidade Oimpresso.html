◆ A identidade do Oimpresso

O fluxo é um círculo, não uma linha.

Norte é a linguagem visual do Oimpresso. A tese é uma só: cada módulo é uma costura pro próximo — nada é redigitado, nada vive em silo. A identidade existe pra deixar isso óbvio na tela.

Princípios

Quatro regras que fazem a identidade ser sistema, não enfeite.

01

Costura, não silos

Todo módulo aponta pro próximo. A passagem entre etapas — a costura — é o herói visual: setas, fluxos e o ciclo que se fecha.

02

Cor é navegação

Cada módulo tem uma cor fixa. A Larissa sabe onde está pelo tom da tela, não pelo texto. Cor carrega significado — nunca é decoração.

03

Densidade com calma

O balcão precisa de muita informação numa tela. Neutros quentes, hierarquia tipográfica forte e espaço — denso sem ser sufocante.

04

Uma identidade, dois climas

O noturno conta a história do produto (marca, North Star). O claro é onde se trabalha. Mesmos tokens, mesmo roxo — duas peles.

Cor

O roxo é a âncora. A cor-por-módulo é a assinatura.

O roxo oklch(0.55 0.15 295) já é o equity do Oimpresso — fica. A novidade é o sistema de cor-por-módulo: oito hues no mesmo lightness e chroma, então nenhum grita mais que o outro, e cada um é dono de um módulo.
