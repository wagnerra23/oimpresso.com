Características mínimas — extraídas dos módulos do repo

  • NFC-e / NF-e (Modules/NfeBrasil): certificado A1/A3, emissão manual, reenvio DANFE por e-mail, download PDF, manifestação destinatário, regras NCM por estado.
  • NFS-e (Modules/NFSe): emissão via Sistema Nacional (LC 214/2025), ativada junto com NfeBrasil.
  • MDF-e: módulo ainda não existe no repo — entra na tela como backlog explícito pra não criar expectativa indevida.
  • Disponibilidade dinâmica: NF-e exige CNPJ no cliente; NFS-e só aparece se houver item tipo "serviço"; NFC-e é default pra consumidor final.
