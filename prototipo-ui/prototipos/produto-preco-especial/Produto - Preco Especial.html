F1 — Aba “Preço especial” · modelo regra + exceção

A lista de preço é uma regra (%), não uma grade de digitação. A matriz mostra o preço efetivo calculado; digitar numa célula cria uma exceção. Uma lista por vez. Padrão convergente de Shopify B2B · Tiny/Olist · Bling · Odoo — ver memory/sessions/2026-07-16-arte-grade-variacao-lista-preco.md.

  • Clique nas listas (Varejo / Atacado / Revenda) — o preço recalcula pela regra de cada uma.
  • Digite numa célula → vira exceção (negrito + tarja + contador na aba). O ↺ desfaz.
  • Mude o % da regra → tudo que é herdado recalcula; exceção não se mexe.
  • Vermelho-G não existe (célula hachurada) — combinação desmarcada na geração.
  • Teclado: Tab anda coluna · Enter pula linha (padrão Cin7/Lightspeed).
