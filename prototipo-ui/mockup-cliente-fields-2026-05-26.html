Os 7 gaps que essa proposta resolve:
  1. Background do input — A: transparente (some no card) · B/C: cinza claro (destaca "preencha aqui")
  2. Font-size — A: 14-16px · B/C: 13px (compacto profissional)
  3. Cor do label — A: preto principal (compete com valor) · B/C: cinza dim (hierarquia visual clara)
  4. Label size — A: 12px · B/C: 11.5px
  5. Focus ring — A: ring/50 (azul opaco) · B/C: accent-soft (azul suave, mais elegante)
  6. Marker opcional — A: span muted · B/C: small text-mute 10.5px (mais sutil)
  7. Radio — A: nativo browser · B/C: pill on=accent-soft (acabamento pro)
